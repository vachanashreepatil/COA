Core Components & Their Functions

Arithmetic Logic Unit (ALU)

The ALU is the heart of the CPU, responsible for performing:

  • Arithmetic operations (addition, subtraction, multiplication, division)
  • Logical operations (AND, OR, NOT, XOR)
  • Bit shifting operations
  • N-BIT ALU: N-bit ALU describes an ALU that can operate on operands of size 'n' bits. Some popular ALUs are 4-bit ALU, 8-bit ALU, 16-bit ALU, 32-bit ALU, etc.

It consists of combinational logic circuits that perform these operations on binary numbers. The ALU takes two operands as input and produces a result based on the control signals it receives from the control unit.

Registers

Registers are high-speed storage locations within the CPU that temporarily hold:

  • Data being processed
  • Intermediate results
  • Control information

Different types of registers serve different purposes:

  • General Purpose Registers: Store temporary data during processing
  • Accumulator: Stores results of arithmetic and logical operations
  • Status Register: Contains flags indicating various CPU states
  • Instruction Register: Holds the current instruction being executed

Program Counter (PC)

The Program Counter is a special register that:

  • Holds the memory address of the next instruction to be executed
  • Automatically increments after each instruction fetch
  • Can be modified by jump and branch instructions

The Program Counter is essential for sequential program execution and control flow. It works in conjunction with the instruction fetch cycle to ensure proper program execution sequence. When a program starts, the PC is initialized to the starting address of the program, and it updates continuously as the program executes.
